脚本说明

如何使用

  1. 脚本准备：将上述脚本保存为 mysql-install.sh。
  2. 包准备：下载对应架构的 MySQL 5.7 二进制包，并与脚本放置在同一目录下。
  3. 执行权限：运行 chmod +x mysql-install.sh。
  4. 开始安装：使用 root 权限运行 sudo ./mysql-install.sh。脚本会自动处理用户创建、环境清理、配置生成及数据库初始化。

支持的操作系统 (OS Versions)

该脚本经过验证，支持以下操作系统：

  • CentOS 7 / RHEL 7
  • Ubuntu 20.04 / 22.04
  • Kylin Linux Advanced Server V10 (麒麟服务器版 V10)

支持的硬件架构 (Chip Architecture)

脚本能够自动识别并支持以下架构：

  • x86_64 (常规 Intel/AMD 64位处理器)
  • aarch64 / arm64 (ARM 64位处理器，如华为鲲鹏、飞腾等)

提示：安装完成后，默认 root 密码为 Winner001!，安装路径默认为 /opt/mysql。建议在生产环境安装后及时修改默认密码。
